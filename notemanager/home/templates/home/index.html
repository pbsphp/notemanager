Тестовое задание "Менеджер заметок"

Web-приложение для хранения заметок. Каждая заметка имеет заголовок, содержимое, категорию и может быть помечена как “избранная”. Интерфейс приложения должен позволять выполнять поиск/сортировку заметок пользователя по различным критериям.

Пользователи

Админка стандартная: /admin/
Логин: admin
Пароль: sweetroll
Регистрация: /profiles/register/
После регистрации необходимо повторно ввести логин и пароль.
Вход: /profiles/login/
Можно поспользоваться готовым пользователем (testuser : 123)
После авторизации можно разлогиниться используя кнопку в правом верхнем углу (Logout).

Заметки

Заметки можно создавать, читать, модифицировать и удалять, используя соответствующие кнопки наверху таблицы. Пользователь может работать только со своими заметками. Каждая заметка содержит заголовок, содержимое, дату и время создания, категорию, отметку "избранная". Некоторые заметки также имеют публичный uuid, используя который можно посмотреть содержимое заметки в виде html-кода. Категории настраиваются в админке. Таблица с заметками поддерживает сортировку и фильтры используя встроенные возможности extjs 3.4.

Управление заметками

Пользователь может модифицировать заголовок, категорию и текст заметки используя соответствующую форму. Для публикации заметки необходимо кликнуть ячейку Public в таблице, после чего появится публичная ссылка на текст заметки. Для отмены публикации необходимо кликнуть на ячейку еще раз. Для изменения ответки "избранная" нужно кликнуть на ячейку "Favourite". Дата создания заметки может изменяться только администратором.

Взаимодействие с сервером

Практически для всех операций по управлению заметками используется ajax. Тестовое приложение крутится на бесплатном тарифе Heroku, а потому не отличается быстродействием. Если кнопка не работает, скорее всего стоит подождать несколько секунд.

Серверная сторона

В качестве БД используется SQLite3, в качестве БД на heroku - PostgreSQL. Приложение разрабатывалось под Python3, а потому под второй версией может работать некорректно. А может и корректно.
